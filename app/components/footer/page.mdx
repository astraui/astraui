# Footer

A semantic and responsive Footer system composed of three building blocks: `Footer`, `FooterSection`, and `FooterContent`. Each component accepts `className` for styling and `children` for flexible nested layout.

<div className="my-8 flex flex-col gap-4">
  <ComponentWrapper>
    <Footer>
      <FooterSection title="Acme Inc.">
        <FooterContent>Optimize, empower</FooterContent>
      </FooterSection>
      <FooterSection title="Resources">
        <FooterContent>Company info</FooterContent>
      </FooterSection>
      <FooterSection title="Connect">
        <FooterContent>GitHub</FooterContent>
      </FooterSection>
    </Footer>
  </ComponentWrapper>
</div>

## Installation

Add the Footer component from Astra UI using your favorite package manager for quick setup.

<div className="my-8 flex flex-col gap-4">
  <CommandLine command="npx astui@latest add Footer" />
  <CommandLine command="pnpm dlx astui@latest add Footer" />
</div>
