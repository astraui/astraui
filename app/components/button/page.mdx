# Button

A sleek, customizable button component with variants, sizes, and icons for consistent UI design.

## Installation
Install the button component from Astra UI using your package manager for quick integration.

<div className="my-8 flex flex-col gap-4">

    <CommandLine command="pnpm dlx astui@latest add button" />

## Usage
Triggers an action or event when clicked; customizable for labels, icons, and interaction states.

<div className="my-8 flex flex-col gap-4">
    <CommandLine language='tsx' command='import Button from "@/components/ui/Button"' />
    <CommandLine language='tsx' command='<Button variant="link" size="lg" disabled="true" onClick={handleClick()} className="font-bold">Button</Button>' />
