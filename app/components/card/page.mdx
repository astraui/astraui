# Card

A minimalist and versatile card component that supports images, headers, content, and footers elegantly.

<div className="my-8 flex flex-col gap-4">
  <ComponentWrapper>
    <div className="flex items-center gap-4">
      <Card>
        <CardImage src="/desktop.png" width={400} height={200} alt="Example" />
        <CardHeader>
          <CardTitle>Card Title</CardTitle>
          <CardDescription>
            This is a description of the card content
          </CardDescription>
        </CardHeader>
        <CardContent>
          <div>Main content goes here</div>
        </CardContent>
        <CardFooter>
          <Button>Action</Button>
        </CardFooter>
      </Card>
    </div>
  </ComponentWrapper>
</div>

## Installation

Install the Card component quickly using your favorite package manager with these simple commands.

<div className="my-8 flex flex-col gap-4">
  <CommandLine command="npx astui@latest add Card" />
  <CommandLine command="pnpm dlx astui@latest add Card" />
</div>
