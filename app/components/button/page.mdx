# Button

A sleek, customizable button component with variants, sizes, and icons for consistent UI design.

<div className="my-8 flex flex-col gap-4">
  <ComponentWrapper>
    <Button>Button</Button>
  </ComponentWrapper>
</div>

## Installation

Install the button component from Astra UI using your package manager for quick integration.

<div className="my-8 flex flex-col gap-4">
  <CommandLine command="npx astui@latest add button" />
  <CommandLine command="pnpm dlx astui@latest add button" />
</div>

## Usage

Triggers an action or event when clicked; customizable for labels, icons, and interaction states.

<div className="my-8 flex flex-col gap-4">
  <CommandLine
    language="tsx"
    command='import Button from "@/components/ui/Button"'
  />
  <CommandLine
    language="tsx"
    command='<Button variant="link" size="lg" disabled={true} onClick={handleClick} className="font-bold">Button</Button>'
  />
</div>

## Variants

Versatile button variants for primary actions, secondary prompts, destructive events, outlines, and subtle interactions.

<div className="my-8 flex flex-col"></div>

### Primary

The primary button variant emphasizes key actions with bold styling for maximum user focus.

<div className="my-8 flex flex-col gap-4">
  <ComponentWrapper>
    <Button variant="primary">Button</Button>
  </ComponentWrapper>
  <CommandLine
    language="tsx"
    command='<Button variant="primary">Button</Button>'
  />
</div>

### Secondary

The secondary button variant is used for less emphasized actions with a medium level of focus.

<div className="my-8 flex flex-col gap-4">
  <ComponentWrapper>
    <Button variant="secondary">Button</Button>
  </ComponentWrapper>
  <CommandLine
    language="tsx"
    command='<Button variant="secondary">Button</Button>'
  />
</div>

### Outline

The outline button variant is used for neutral actions with a subtle border emphasis.

  </ComponentWrapper>
  <CommandLine

### Ghost

The ghost button variant is used for minimal emphasis actions with no background or border.
